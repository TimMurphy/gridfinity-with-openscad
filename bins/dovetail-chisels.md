# Dovetail Chisels Holder

Solid Gridfinity bin with cylindrical cutouts for 3mm, 6mm, and 12mm dovetail chisels.

- Blade widths: 3mm, 6mm, 12mm
- Overall Chisel length: 225mm
- Blade length: 60mm
- Tapper from handle to chisel
    - Length 64mm
    - Handle end width 17mm
    - Blade end width 8.3mm
- Handle
    - Length: 255 - 60 - 64mm
    - Top width: 25mm wide for 11mm length 20mm width
    - Bottom width: 20mm
    - Midpoint width: 21.5mm
- Finger Relief
    - Add a recess at midpoint of handle for finger and thumb to grab chisel.
- Make the blades lie flat within 5mm recess
